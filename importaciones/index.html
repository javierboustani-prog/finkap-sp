↔ Cómo se conecta con Finkap

El costo real landed de cada origen (FOB + flete + nacionalización + costo de financiación) sale de este módulo y alimenta el pricing-engine / simulador. Así el precio de venta refleja lo que de verdad te cuesta traer cada café — no una estimación. El embarque manda; el precio obedece.
